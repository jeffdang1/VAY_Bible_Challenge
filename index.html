Features & Instructions

Quick guide to every feature. Scroll to read, or download the files below (same folder as this game). If a click opens the file in the browser instead of saving, use your browser’s “Save as” or run the game from a simple local web server.

Import template: Google Sheets template (two rounds) Download .csv Download .json Import guide (.md) Features & instructions (.md)

Google Sheets is the main template: two full rounds, optional Vietnamese columns. Make a copy or use File → Download → CSV, then import here. Vietnamese (CategoryVI, ClueVI, etc.) is optional — leave blank if English only.

1. Game setup

Timer

Seconds per question (5–120). Default 30. Starts when a clue opens; stops when you click Reveal Answer.

Daily Doubles

Choose how many (0–10), placed randomly. Hide keeps them looking like normal values until picked. Player wagers (up to score or 500), then answers; right/wrong adds or subtracts the wager.

Two rounds

Loads the two-rounds template. Round 2 doubles displayed dollar values and resets scores to 0. After Round 1 Final Jeopardy you see ROUND 2. After Round 2, a table shows Round 1, Round 2, and total per team.

Players

Use Add player for up to eight teams (minimum two).

Team buzzer

On by default. First buzz plays a high beep and starts the answer timer (default 10s, adjustable in settings). A lower beep plays when answer time ends. Continue question timer (if enabled) lets other teams buzz again.

Final Jeopardy wagers

On the wager screen, click Start 30s Wager Timer while teams write wagers. A lower beep plays when time is up.

Import questions

Main template: Google Sheets — two rounds (6 categories × 5 questions per round, both finals). Edit online, then File → Download → Comma Separated Values (.csv) and import. Or download the matching .csv / .json from the links above. Vietnamese is optional — CategoryVI, ClueVI, AnswerVI, and ExplanationVI can stay filled, be cleared, or be removed; only English is required. When Vietnamese is present it appears below English in yellow. Enable Two Rounds, then Choose file → Import.

Editor

Show Editor to edit board categories and clues. Final Jeopardy is edited separately at the bottom (no CSV needed). Changes last until import or restart.

Start game

Begins play with current options and questions.

2. During the game

Board

Click a value to open a clue. Cleared cells are gray. Round 2 shows doubled dollar amounts.

Who’s answering

Select the player button before you mark Correct or Incorrect.

Scoring

Correct / Incorrect — adjust the selected player’s score (or Daily Double wager). Pass — no one got it; no points; clue still clears.

Clue window

Reveal Answer shows the response and stops the timer. Close exits without scoring.

Daily Double flow

Enter wager → Next → read clue → score as usual.

Skip to Final Jeopardy

Ends the board early and starts Final Jeopardy (not available after that round’s Final is finished).

3. Final Jeopardy

Intro → wagers → clue → Reveal Answer → mark each player correct or wrong → Close. Max wager is current score or 500. Correct adds your wager; incorrect subtracts it (same rule as Daily Double).

4. Settings & stats

Settings (gear icon) — edit scores, names, timer; open statistics; Restart returns to setup. Statistics (chart icon) — live overview, speed, by category, per team.

What you can do

Two to eight teams, custom names, built-in or imported questions, timer and Daily Double options, optional two-round game with combined final scores, in-game adjustments, and full play statistics.
